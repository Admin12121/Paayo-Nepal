# Tourism Frontend

Next.js 16 frontend application. See the root [README.md](../README.md) for full documentation.

## Development

```bash
bun install
bun run dev
```

## Database

```bash
bun run db:push      # Push schema
bun run db:studio    # Open Drizzle Studio
bun run db:generate  # Generate migrations
bun run db:migrate   # Run migrations
```
